# TradeMate
Мобильное приложение, которое упростит работу компаний, поставляющих товары в магазины.

Для мерчендайзеров:
- Создание заявок о необходимых для магазинов товарах и отправка их на электронную почту операторов
- Хранение фотографий в облаке

Для руководителей:
- Контроль работы сотрудников
- Управление персоналом
